# Lab Virtual Wireless Hacking

> [!note]
> Dokumentasi dipindahkan ke: [https://github.com/fixploit03/Lab-Virtual-Wireless-Hacking/wiki](https://github.com/fixploit03/Lab-Virtual-Wireless-Hacking/wiki)

## Instalasi

```bash
sudo apt-get update
sudo apt-get install macchanger hostapd dnsmasq wpasupplicant iw freeradius dhclient
git clone https://github.com/fixploit03/Lab-Virtual-Wireless-Hacking
cd Lab-Virtual-Wireless-Hacking
chmod +x setup.sh
sudo ./setup.sh
```

## Penggunaan

### Menjalankan Lab

```
sudo ./start-lab.sh
```

### Menghentikan Lab

```
sudo ./stop-lab.sh
```
