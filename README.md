## Instalasi

```bash
git clone https://github.com/fixploit03/Lab-Virtual-Wireless-Hacking
cd Lab-Virtual-Wireless-Hacking
chmod +x setup.sh
sudo ./setup.sh
```

## Penggunaan

### Menjalankan Lab

```
sudo ./start-lab.sh
```

> [!NOTE]
> Menjalankan lab butuh waktu sekitar:
>
> ```
> real  7m52,015s
> user  0m0,018s
> sys   0m0,030s
> ```

### Menghentikan Lab

```
sudo ./stop-lab.sh
```
