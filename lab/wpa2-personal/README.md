# Lab Wi-Fi WPA2-Personal

![](https://github.com/fixploit03/Lab-Virtual-Wireless-Hacking/blob/main/img/lab%20wpa2.png)

## Konfigurasi Interface
- `wlan0`: Attacker
- `wlan1`: AP & DHCP Server
- `wlan2` - `wlan4`: STA

## Instalasi

Kloning repositori:

```
git clone https://github.com/fixploit03/Lab-Virtual-Wireless-Hacking
cd Lab-Virtual-Wireless-Hacking/lab/wpa2-personal
```

Beri izin eksekusi:

```
chmod +x start-lab.sh stop-lab.sh
```

Jalankan lab:

```
sudo ./start-lab.sh
```

Stop lab:

```
sudo ./stop-lab.sh
```
